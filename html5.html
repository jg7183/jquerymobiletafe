Q2. Briefly explain the purpose of HTML?

The purpose of HTML is to in a sense draw up a skeleton of a webpage with elements and tags, these elements can then be manipulated using other languages like CSS/Javascript in order to make them look nice and give them functionality

Q3. Briefly explain the purpose of CSS?

CSS or Cascading Style Sheets, are a seperate sheet linked to the HTML document, using the CSS language you can thene manipulate and style the HTML elements

Q4. What is Javascript?

Javascript is the scripting language of the internet, it gives HTML functionality (i.e clicking a button causing a pop up or submiting a form somewhere)
